# Run

- Make sure you have vlc (and add it to path if on windows)
- Make sure the terminal is large (80col) and long (50col) enough
- Run cli-rhythm

# Compiling

## Dependencies

- make
- gcc

## Linux

- `make`

## Windows

- have gcc and make in your path
- `make`
